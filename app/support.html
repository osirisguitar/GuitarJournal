Bugs

The quickest way to get my attention is to post an issue to Github. If you don’t have a Github account, email me at info@osirisguitar.com.

New features

There are a number of features I've considered for coming versions of the OG Journal. Some of them are already on Github, others are:

  • Punch clock-style starting and stopping sessions.
  • Audio and/or video recording for sessions
  • More stats
  • Excel export of sessions to let users make pivot diagrams or other detailed analysis
  • Login and sharing on other social networks (Twitter is next).
